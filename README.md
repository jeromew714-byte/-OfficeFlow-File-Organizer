# -OfficeFlow-File-Organizer
A simple Windows utility that organizes messy folders into clean, labeled directories. Runs locally. No cloud. No tracking.
## Overview
OfficeFlow is a lightweight Windows file-organization utility designed for local use.
It does not install software, transmit data, or require internet access.

## Key Characteristics
- Runs locally on Windows
- No cloud services
- No telemetry or tracking
- Files never leave the machine
- Source code is publicly available for review

## Typical Use Cases
- Office document cleanup
- Church or nonprofit record organization
- CAD and project file sorting
- Personal productivity workflows
